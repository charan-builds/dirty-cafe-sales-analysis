Project Title
Dirty Cafe Sales Data Analysis
Problem Statement
The dataset contains messy cafe transaction data
with missing values, unknown categories,
incorrect data types, and inconsistencies.

Goal:
Clean the dataset and perform Exploratory Data Analysis (EDA)
to derive business insights.
Steps Performed
1. Data Understanding
2. Data Cleaning
3. Missing Value Treatment
4. Datatype Conversion
5. Outlier Analysis
6. Univariate Analysis
7. Bivariate Analysis
8. Correlation Analysis
9. Feature Engineering
10. Feature Selection
Key Insights

Example:

• Total Spent is moderately right-skewed.
• Salad generates the highest revenue.
• Revenue differences are driven mainly by Price Per Unit.
• Average quantity sold is similar across products.
• Location and Payment Method contain large amounts of unknown data.
• Item and Price Per Unit have a strong business relationship.
Libraries Used
Pandas
NumPy
Matplotlib
Seaborn
